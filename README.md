# 🎬 Movie Recommendation App

A content-based Movie Recommendation System built with **Python**, **Streamlit**, and **TMDB API**. Search for your favorite movie and instantly receive five similar movie recommendations along with their posters.

## 🚀 Live Demo

🔗 **Deployed App:** **https://movie-recommendation-app-arsu13.streamlit.app/**

---

## 📌 Features

* 🔍 Search movies by name
* 🎯 Content-based recommendation system
* 🖼️ Movie posters fetched using TMDB API
* ⚡ Fast and interactive Streamlit interface
* ☁️ Model downloaded automatically from Google Drive
* 📱 Responsive layout

---

## 🎯 How It Works

1. Search for a movie using the search bar.
2. Select the matching movie from the suggestions.
3. Click **Recommend Movies**.
4. The application displays five similar movies with their posters fetched from TMDB.

---

## 📚 Dataset

This project uses a movie dataset containing movie titles and TMDB IDs. The recommendation model is generated using content-based filtering and is downloaded automatically from Google Drive during the first run.
